Displays the current position of the International Space Station live, in terminal.

Made with  
https://github.com/yaronn/blessed-contrib  
http://api.open-notify.org/

# Usage
```{r, engine='bash', count_lines}
npm install
npm run start
```
```{r, engine='bash', count_lines}
<Ctrl-C to exit>
```
